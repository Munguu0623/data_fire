# Улаанбаатарын гал түймрийн эрсдэлийн прогноз

Хороо түвшинд, сар бүрээр гал гарах эрсдэлийг урьдчилан тооцоолж,
Онцгой байдлын албаны урьдчилсан сэргийлэх сургалтыг **хаана, хэзээ**
чиглүүлэхэд туслах шийдвэр дэмжих систем.

XGBoost машин сургалтын 3 загвар (ил гал / цахилгаан / яндан-цонолт) +
интерактив веб дашборд.

## Ажиллуулах

```bash

python scripts/01_population.py        # хүн ам цэвэрлэх
```

## Арга зүй

**Нэгж:** хороо × сар (204 хороо × 132 сар = 26,928 мөр panel).
Гал гараагүй хороо-сар автоматаар "сөрөг жишээ" болж XGBoost-д хэрэгтэй
тэнцвэрийг үүсгэдэг.

**Features (бүгд leakage-гүй, зөвхөн өнгөрсөн дата):**
- Түүхэн lag: өмнөх 1/2/3/12 сарын галын тоо
- Rolling нийлбэр: сүүлийн 3/6/12 сар
- Хорооны түүхэн дундаж (cummean)
- Улирлын: тухайн хорооны тухайн сарын түүхэн эрсдэл (seasmonth)
- Цаг хугацаа: сарын sin/cos, жилийн тренд
- Хүн ам (log)

**Сургалт/тест хуваалт:** 2015-2023 сургалт, 2024-2025 тест
(цаг хугацааны дараалал хадгалсан — санамсаргүй хуваалт ХИЙГЭЭГҮЙ).

**Class imbalance:** `scale_pos_weight` ашигласан (гал ховор үйл явдал).

---

## Үр дүн (тест: 2024-2025)

| Шалтгаан | ROC-AUC | PR-AUC | Дүгнэлт |
|----------|---------|--------|---------|
| Яндан/цонолт | **0.84** | 0.31 | Сайн — улирлын тогтмол хэв маяг |
| Ил гал | 0.65 | 0.47 | Дунд — хүний зан үйлээс хамаарна |
| Цахилгаан | 0.62 | 0.32 | Дунд — байршлын дохио сул |

**Тайлбар:** Яндан/цонолт хамгийн найдвартай (өвөл тогтмол оргилдог).
Ил гал, цахилгаан хороо түвшинд сул — учир нь хүний зан үйлээс хамаарч
байршлын тогтмол хэв маяг сул.

## Дараагийн алхам — нарийвчлал нэмэх

Загварын ялгах чадварыг сайжруулахад дараах дата нэмэх шаардлагатай.
Архитектур модульчлагдсан тул эдгээрийг нэг баганаар нэмж болно.

1. **Хорооны хилийн GeoJSON** (хамгийн чухал) — энэ нэг файл дараах бүгдийг нээнэ:
   - `ger_points` (67,646 гэрийн цэг) → хороо бүрийн **гэрийн нягтрал**
   - OSM landuse/building → газар ашиглалт, барилгын нягтрал
   - Эх сурвалж: overpass-turbo, chrismfree.com/mongolia-gis-data,
     эсвэл ҮСХ (international@nso.mn)

2. **Цаг агаар** — сарын дундаж температур, салхи, чийг, хур тунадас
   - Open-Meteo Archive API (үнэгүй, key шаардахгүй)
   - Ил галд онцгой чухал (хуурайшил, салхитай шууд холбоотой)

3. **Нийгэм-эдийн засаг** — хороогоор орлого, ажилгүйдэл (ҮСХ)

тодорхой гал гарахыг батлахгүй. Хүний мэргэжлийн дүгнэлтийг орлохгүй.
